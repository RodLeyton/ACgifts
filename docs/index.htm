We have a new toy for AC fans!

Whether you trade gifts just on your main or have many fuel rings to maintain, tracking gifts can become difficult and time consuming.
Speadsheets can do the job sure, but quickly get out of hand and bloated especially as your game count increases.

Introducing ACgifts

A simple app that keeps count of gifts sent and recieved, sorted by groups (such as fuel rings)

Neighbors can be imported via CSV file or directly entered and edited in app.

Free and open source

The project is hosted on GitHub and contributors are very wellcome. Contact me via the forum to request access to the repo.
The published files are signed by my own certificate, so you know only I can make updates to the installed app no matter who adds features.
But the best feture of open source code is you can view it, compile it and modify it to your liking, and I encourage you to do so!
This repo compiles in Visual Studio 2022 community (free) without a single warning or reccomendation.

Quick and easy setup

Packaging and deployment using Microsoft ClickOnce technology enables simple install without requiring elevated priviledges (non-admin) along with automatic and rapid updates.
Microsoft .NET6 is required, but you will be directed during install if your PC does not have it already. (if not find it here)
Be forewarned you will recieve multiple warnings before install because I created my own certificates without paying a 'trusted authority' that only confirms my identity, not how trustworthy the code signed is.
